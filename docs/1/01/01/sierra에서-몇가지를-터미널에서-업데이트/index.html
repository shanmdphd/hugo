Sierra에서 몇가지를 터미널에서 업데이트

0001/01/01

Ruby, Gem, Brew를 문득 업데이트 하고 싶어져서 하나씩 실행하다가 혹시 다음에 또 필요할까봐 기록해 놓습니다.

Ruby update

curl -L https://get.rvm.io | bash -s stable –ruby
echo rvm_autoupdate_flag=2 >> ~/.rvmrc
rvm list known
rvm install ruby-2.4.0
rvm use ruby-2.4.0

Gem update

gem update

Brew update
